##Gallería de Imágenes demo con AngularJS, Node Server y Mongodb


Gallería de imágenes simple y basada en NodeJs, AngularJS y MongoDb

Utilizando Bootstrap 4 y Bootstrap-table


###Install

```
npm i
```

###Start NodeJS Server

```
npm start
```
